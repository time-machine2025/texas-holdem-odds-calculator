# 德州扑克胜率计算器

一个纯前端的德州扑克胜率计算器。打开 HTML 文件即可使用，不需要后端服务，也不需要安装依赖。

## 功能

- 设置玩家总数，支持 2 到 10 人桌
- 录入自己的两张初始手牌
- 实时更新翻牌、转牌、河牌
- 自动计算胜率、平局率、失败率和摊牌权益
- 支持三档模拟精度：快速、均衡、细致
- 支持随机手牌、随机公牌、清空公牌和全部清空
- 适配桌面端和移动端浏览器

## 使用方式

直接用浏览器打开：

```text
texas-holdem-odds.html
```

页面打开后：

1. 设置玩家总数。
2. 点击牌面录入自己的两张手牌。
3. 在牌局进行中切换到公牌区域，依次录入翻牌、转牌、河牌。
4. 页面会在每次输入后自动重新计算胜率。

## 计算说明

本项目使用蒙特卡洛模拟计算胜率：

- 根据已知手牌和公牌排除不可用牌。
- 随机补全剩余公牌和其他玩家手牌。
- 对每位玩家的 7 张牌取最佳 5 张牌型。
- 重复模拟多次后统计胜、平、负概率。

因为模拟依赖随机抽样，所以在公牌较少或玩家较多时，每次计算结果可能会有轻微波动。需要更稳定的结果时，可以选择更高的计算精度。

## GitHub Pages 部署

上传到 GitHub 后，可以通过 GitHub Pages 发布：

1. 进入仓库的 `Settings`。
2. 打开 `Pages`。
3. 在 `Build and deployment` 中选择 `Deploy from a branch`。
4. 选择 `main` 分支和根目录 `/root`。
5. 保存后等待 GitHub 生成访问链接。

如果希望访问仓库主页时直接打开计算器，可以把 `texas-holdem-odds.html` 重命名为 `index.html`。
